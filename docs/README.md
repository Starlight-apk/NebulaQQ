## 🌟 特性亮点

1. **插件系统** - 强大的插件机制，支持依赖管理和热重载
2. **模块系统** - 完全模块化的架构，按需加载，热插拔
3. **主题系统** - 支持自定义主题，打造个性化机器人
4. **高性能** - 优化的连接池和缓存机制
5. **TypeScript** - 完整的类型定义，优秀的开发体验

## 📦 核心模块

| 模块 | 说明 |
|------|------|
| `@nebulaqq/core` | 核心模块 - 事件系统、消息处理 |
| `@nebulaqq/plugin-sdk` | 插件 SDK - 插件开发工具包 |
| `@nebulaqq/module-sdk` | 模块 SDK - 模块开发工具包 |
| `@nebulaqq/theme-sdk` | 主题 SDK - 主题开发工具包 |
| `@nebulaqq/adapter-onebot` | OneBot 协议适配器 |
| `@nebulaqq/network` | 网络模块 - HTTP/WebSocket |
| `@nebulaqq/utils` | 工具函数库 |
